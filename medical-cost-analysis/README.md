# Medical Cost Analysis

This project analyzes a medical insurance dataset to investigate demographic and lifestyle factors associated with healthcare costs.

## Dataset

- insurance.csv

## Objectives

- Investigate the relationship between age and medical costs.
- Analyze how BMI influences insurance charges.
- Evaluate the impact of smoking status on healthcare expenses.
- Compare obesity prevalence across regions.

## Skills Demonstrated

- Python
- pandas
- NumPy
- Matplotlib
- Data cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Data visualization
