# TSA Airport Claims Analysis

This project analyzes TSA airport insurance claims data as part of my Python Data Analysis coursework.

## Dataset

- tsa_claims2.csv
- GlobalAirportDatabase.csv

## Questions Explored

- What is the most common type of insurance claim?
- Which airport claim sites receive the most claims?
- What is the typical claim amount?
- What is the overall claim approval rate?
- Which airports have the most claims?
- Has the total close amount increased or decreased over time?

- Python
- pandas
- NumPy
- Matplotlib
- GeoPandas
- Data cleaning
- Datetime analysis
- Exploratory Data Analysis (EDA)
