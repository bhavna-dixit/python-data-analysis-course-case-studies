# Python Data Analysis Case Studies

This repository contains end-to-end data analysis case studies completed as part of the **Python Data Analysis** course at the **University of California, Santa Cruz (UCSC) Extension**.

These projects demonstrate the complete analytical workflow from data cleaning and exploratory analysis to visualization and interpretation using Python.

---

## Skills Demonstrated

- Python
- pandas
- NumPy
- Matplotlib
- Data cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Data visualization
- GroupBy and aggregation
- Datetime analysis
- Geospatial analysis (GeoPandas)

---

## Projects

### Medical Cost Analysis

Analyze demographic and behavioral factors associated with medical insurance charges.

Key questions explored:

- How does age relate to medical costs?
- How does BMI influence insurance charges?
- How does smoking status affect healthcare costs?
- Which region has the highest obesity percentage?

**Skills**
- Data cleaning
- Feature engineering
- Visualization
- Statistical summaries

---

### TSA Airport Claims Analysis

Analyze TSA airport insurance claims to identify operational trends and claim patterns.

Key questions explored:

- Most common claim types
- Claim approval rates
- Airport-level claim statistics
- Trends in claim amounts over time
- Geospatial visualization of airport claims

**Skills**
- Data cleaning
- Datetime analysis
- Aggregation
- Geospatial visualization
- Trend analysis
